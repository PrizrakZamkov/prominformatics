Бесплатный хостинг для сайтов

neocities.org  
 

• https://neocities.org  был запущен 24 мая 2013 года Кайлом Дрейком. Является одним из лучших в рейтингах бесплатных хостов и наиболее приятен по отзывам пользователей.



Плюсы:
 1. Возможность создавать сайт из своих HTML-страничек, а не из шаблонов и некоего конструктора. То есть для создания страниц своего сайта приветствуется использование HTML, CSS и Javascript.


       2. Бесплатность и отсутствие рекламы. Деньги на поддержание существования хостинга предполагается получать от тех пользователей, которые добровольно захотят перейти на платный тариф. Также создатели хостинга рассчитывают на пожертвования от благотворителей.

  
       3. Приверженность автора проекта идеологии открытых исходных кодов (open source): https://github.com/neocities.

 
       4. Весь сайт можно скачать обратно к себе на компьютер в ZIP-архиве.
 

    
Минусы: 1. Не поддерживается доступ по FTP.
Кнопка загрузки файлов через веб-интерфейс позволяет загрузить сразу множество файлов, но не позволяет загрузить папку с файлами. Если работа идет только через веб-интерфейс, папки сначала придется создать, а только затем можно загружать туда файлы.
Вместо доступа по FTP пользователю предлагается воспользоваться API сервиса (https://neocities.org/api).

    
        2. Данный хостинг держится на одном человеке — Кайле Дрейке.
